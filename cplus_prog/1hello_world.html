C++ Program to Print "Hello World"



Source code

//C++ Program to display "Hello World"

#include<iostream>
using namespace std;
				
//main() function: where execution of program begins.
int main() 
{

     //prints "Hello World"
     cout << "Hello World";
				
     return 0;
}

Output

Hello World

Explanation of C++ Hello World Program

1. Comment line: // C++ Program to display "Hello World"

As the name implies, comments are simply text written by programmers during the creation of code. It is completely ignored by the C++ compiler. However, they should be relevant to the code and have some context so that when someone else looks at your code, they can understand what you did just by reading your comment.


Two different ways to give comment line:

a) // This is used to represent the single line comment
b) /* This is used to represent the multiple line comment
suitable for more than one line of instruction */

2. Header file: #include < iostream >

This statement instructs the compiler to include the iostream file in the output. This file includes input/output functions that can be used in our software. The preprocessor directive #include is used to include files in our software.

3. using namespace std

using namespace std means that we can use names for objects and variables from the standard library.

4. int main()

A valid C++ program must have the main() function. The curly braces indicate the start and the end of the function.
The execution of code beings from this function.

5. cout

prints the content inside the quotation marks. It must be followed by << followed by the format string. In our example, "Hello World" is the format string.
; is used to indicate the end of a statement.

6. return 0

The return 0; statement is the "Exit status" of the program. In simple terms, the program ends with this statement
